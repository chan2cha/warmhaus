# Product Overview

## Summary

Warmhaus Lead OS는 인테리어 상담과 견적 문의를 온라인에서 접수하고, 운영자가 고객 리드를 검토한 뒤 상담 일정을 협의하고 안내 메시지를 발송하기 쉽게 만든 내부 운영 서비스입니다.

이 서비스는 영업 CRM이라기보다, 상담 접수 이후 초기 응대와 일정 조율을 빠르게 처리하기 위한 운영 도구에 가깝습니다.

## Primary Goals

- 고객 문의를 일관된 형식으로 수집한다.
- 공사 범위와 예산 정보를 미리 받아 상담 품질을 높인다.
- 상담 희망 시간을 구조화해 일정 협의를 단순화한다.
- 운영자가 문자 문구를 반복 작성하지 않도록 돕는다.
- 공지와 접수 Rule을 운영자가 직접 관리할 수 있게 한다.

## Users

### 고객

- 브랜드 소개 페이지를 보고 문의를 남기는 사용자
- 전화 상담 또는 대면 상담을 요청하는 사용자
- 공사 범위와 예산, 일정 계획을 입력하는 사용자

### 운영자 / 관리자

- 신규 리드를 확인하는 사용자
- 상담 가능 시간을 조율하는 사용자
- 문자 안내 문구를 복사하거나 전송 링크를 사용하는 사용자
- 공지사항과 접수 Rule을 변경하는 사용자

## Core Modules

### 1. 고객 공개 페이지

- 브랜드 링크 제공
- 홈페이지, 블로그, SNS 연결
- 문의 폼 진입 제공

### 2. 상담/견적 폼

- 고객 기본 정보
- 주소와 면적
- 예산 범위
- 공사 희망 항목
- 가구 관련 요청
- 상담 방식과 희망 시간

### 3. 관리자 인박스

- 리드 목록 조회
- 일정 기준 탭 분류
- 빠른 전화 액션
- 리드 상세 진입

### 4. 일정 협의

- 고객이 제출한 희망 시간 후보 관리
- 상태 변경
- 일정 확정 / 취소 / 변경 요청

### 5. 문자 템플릿

- 상담 제안 템플릿
- 상담 확정 템플릿
- 전화 / 대면 상담에 따른 문구 분기

### 6. 공지 및 Rule 관리

- 고객 폼 상단 공지 수정
- 허용 지역 설정
- 마감 월 설정
- 특정 일자 부분 가능 설정
- 예산 기준 설정

## Scope Boundaries

현재 프로젝트는 다음 범위에 집중합니다.

- 상담 접수
- 초기 분류
- 상담 일정 협의
- 공지와 운영 정책 관리

다음 범위는 아직 본격 구현 상태가 아닙니다.

- 계약/결제
- 견적서 발행
- 실제 문자 API 발송
- 고급 캘린더 운영 화면
- 정식 관리자 인증
